## 2. Executive Pitch (Under 60 seconds)
The **Agentic Supply Chain Resilience Platform** is a multi-agent system built using the **Google GenAI SDK (Agents Feature)** and **Model Context Protocol (MCP)** to automate operational crisis management in supply chains. 

When a disruption strikes the logistics network (e.g. floods, labor strikes, mechanical breakdowns), a group of specialist agents dynamically coordinate to sanitize user prompts, evaluate inventory safety levels, re-sequence truck carriage routes using mathematical heuristics (FFD / Nearest Neighbor TSP), and score KPIs against organizational preferences. Planners get a visual dashboard showing real-time comparative metrics, step-by-step agent chat traces, audit logs, and professional multi-sheet Excel exports.

---

## 3. Technology Stack & Frameworks
1. **Agent Orchestration**: Google GenAI SDK (Agent Client, AgentConfig, Tool Binding).
2. **Standardized Interfaces**: Model Context Protocol (MCP) local stdio JSON-RPC 2.0 registry.
3. **Core Solvers**: First-Fit Decreasing (FFD) bin packing, Nearest-Neighbor Traveling Salesperson (TSP) sequencer.
4. **User Interface**: Streamlit with custom CSS glassmorphism, responsive colors, and interactive HTML hover graphs.
5. **Data Engine**: Pandas and OpenPyXL for multi-sheet workbook generation.

---

## 4. Key Capstone Milestones Met
- **Dual-Mode System**: Merges live LLM conversational agent intelligence with deterministic math solvers.
- **Safety Guardrails**: Strict prompt injection scanning and automatic cost-cap compliance checks.
- **Persistent Preferences Memory**: Active JSON loaders tracking business prioritization profiles.
- **Interactive UX**: Live chatbot for query overrides, dynamic color-coded severity headers, and hover tooltips.
